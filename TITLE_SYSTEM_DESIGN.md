# BRAKE Pro 称号システム 実務設計書

## 概要

BRAKE Proの称号システムは、単なるゲーム的バッジではなく、トレーダーの**規律・損失管理・感情統制・再現性・継続性・資本規模**を総合的に評価し、「本当に格のあるトレーダー」を可視化する仕組みです。

金融エリート感・資本階級感・厳格さ・冷たい洗練さを重視し、日本人が見て「格がある」「強い」「本物っぽい」と感じる名称体系を採用します。

---

## 7層構造

### 1. 通常ランク（Rank）
誰でも一瞬で序列が分かる基本ランク。利益額ではなく、総合的な成熟度を表す。

### 2. 上位クラス（Class）
通常ランクの上に位置する、金融エリート感のあるクラス称号。異なる「強さの型」を表現。

### 3. 別格称号（Special Title）
最上位の伝説称号。BNF到達者など、極めて高い基準を満たした者のみ。

### 4. アセット階級（Asset Tier）
資産規模に基づく別軸ラベル。資本主義の階級感を表現。

### 5. ライト称号（Light Title）
軽中量級の達成感・コレクション性を担う称号。20個前後。

### 6. シークレット称号（Secret Title）
隠し条件・超高難度の伝説的な称号。取得率は極小。

### 7. 裏称号（Hidden Title）
ジョーク・自虐・ブラックユーモア。原則非公開、本人向け。

#### ジェーンストリート・クラス
- **一言での印象**：精密・高効率・スマートさ・無駄のなさ
- **世界観**：
  - 定量的分析に基づく精密なトレード
  - テクニカル精度が極めて高い
  - 無駄のない効率的な取引
  - アルゴリズム的な正確さ
- **強さの型**：
  - テクニカル精度が高い（勝率65%以上）
  - 平均RR比が高い（2.5以上）
  - 取引効率が高い（勝ちトレードの平均利益が大きい）
  - 一撃依存度が低い（複数の小さな勝利の積み重ね）
- **審査期間**：90日
- **必須条件**：
  - 月次勝率 ≥ 65%
  - 平均RR比 ≥ 2.5
  - 取引効率スコア ≥ 85点
  - 一撃依存度 ≤ 10%
  - 最大ドローダウン ≤ -15%
  - ルール遵守率 ≥ 95%
- **除外条件**：
  - 重大違反あり
  - 感情的トレード率 > 20%
  - 資産規模 < 100万円
- **失格条件**：
  - 90日以内に最大DD -30%以上
  - ルール遵守率が90%未満に低下
  - 重大違反が発生
- **主要評価軸**：
  - テクニカル精度（勝率）
  - 取引効率（RR比・平均利益）
  - リスク管理（DD・ボラティリティ）
  - 規律遵守（ルール遵守率）
- **想定取得率**：0.5%
- **公開向きかどうか**：◎ 公開向き（テクニカルスキルの証明）
- **UI表示文言**：「精密で効率的なトレード技術の証」
- **他人からどう見られるか**：「テクニカルが強い」「効率的で無駄がない」「スマートなトレーダー」

## 3. 別格称号（Special Title）

### BNF到達者

最上位の伝説称号。上位クラスよりさらに上に位置する。

#### 基本情報
- **名称**：BNF到達者
- **世界観**：
  - 90日での資産拡大速度の神話
  - スピードの伝説
  - 短期での圧倒的な成長
  - 市場の波を読む能力
  - 伝説的なトレーダーの証
- **取得条件**：
  - 審査期間：90日
  - 最も資産拡大速度が高い者に与える
  - 厳格な補正を入れた評価
  - 上位0.1%未満レベル
  - 乱発されない
- **除外条件**：
  - 重大違反あり
  - ルール遵守率 < 90%
  - 感情的トレード率 > 15%
  - 資産規模 < 50万円
- **失格条件**：
  - 90日以内に最大DD -40%以上
  - 月次勝率が50%未満に低下
  - 重大違反が発生
  - ルール遵守率が85%未満に低下

#### 評価式

```
BNF_Score = (
  (補正後資産成長率 × 0.35) +
  (リスク調整後リターン × 0.25) +
  (ディシプリンスコア × 0.15) +
  (エモーショナルコントロール × 0.15) +
  (継続性スコア × 0.10)
)

ただし、以下の条件で減点：
- 最大DD -20%以上：-10点
- 最大DD -30%以上：-20点
- 最大DD -40%以上：失格
- 一撃依存度 > 30%：-15点
- 感情的トレード率 > 20%：-10点
- ルール遵守率 < 95%：-5点
```

#### 主要評価軸
- 入出金補正後の純資産成長率（35%）
- リスク調整後リターン（25%）
- ディシプリン（15%）
- エモーショナルコントロール（15%）
- 継続性（10%）

#### 必須条件
- 審査期間 ≥ 90日
- 資産拡大速度（補正後） 上位0.1%以内
- リスク調整後リターン ≥ 2.5
- 最大ドローダウン ≤ -20%（推奨：-15%以内）
- ディシプリンスコア ≥ 90点
- エモーショナルコントロール ≥ 85点
- 重大違反ゼロ
- 一撃依存度 ≤ 20%
- 感情的トレード率 ≤ 15%
- 取引数 ≥ 30回
- ルール遵守率 ≥ 95%

#### UI表示文言
- 「90日での資産拡大速度の伝説」
- 「スピードの神話」
- 「BNF到達者」

#### 公開時の見せ方
- 最高峰のステータス表示
- 特別なバッジ・アイコン
- 「伝説」「神話」の表現
- SNS共有時に特別なカード

#### 乱発防止策

- 上位0.1%の厳格な基準
- 複数の評価軸の高水準化

## 4. アセット階級（Asset Tier）

## 評価軸の設計

称号付与の基盤となる6つの評価軸。

### 1. キャピタルグロース（資本成長）

**何を測るのか**：
- 入出金補正後の純資産成長率
- 月次・四半期・年次での成長率
- リスク調整後リターン

**スコア化方法**：
```
CapitalGrowth_Score = (
  (補正後資産成長率 / 目標成長率) × 50 +
  (リスク調整後リターン / 目標RaR) × 50
)
```

**満点**：100点

**どのランク/クラス/称号で重視するのか**：
- チャレンジャー：最重視（35%）
- BNF到達者：最重視（35%）
- ジェーンストリート・クラス：重視（20%）
- ゴールドマン・クラス：重視（20%）

### 2. リスクコントロール（リスク管理）

**何を測るのか**：
- 最大ドローダウン（DD）
- 月次最大損失
- ボラティリティ
- ポジションサイズの適切性

**スコア化方法**：
```
RiskControl_Score = (
  (目標DD / 実績DD) × 40 +
  (月次安定感スコア) × 30 +
  (ボラティリティ管理) × 30
)
```

**満点**：100点

**どのランク/クラス/称号で重視するのか**：
- JPモルガン・クラス：最重視（40%）
- プラチナ：重視（25%）
- ダイヤモンド：重視（25%）

### 3. ディシプリン（規律）

**何を測るのか**：
- ルール遵守率
- チェックフロー実施率
- 冷却タイマー遵守率
- 誓約実行率

**スコア化方法**：
```
Discipline_Score = (
  (ルール遵守率) × 40 +
  (チェックフロー実施率) × 30 +
  (冷却タイマー遵守率) × 20 +
  (誓約実行率) × 10
)
```

**満点**：100点

**どのランク/クラス/称号で重視するのか**：
- ブロンズ～プラチナ：必須
- ディシプリンド称号：最重視

### 4. エモーショナルコントロール（感情統制）

**何を測るのか**：
- 感情的トレード率
- FOMO傾向スコア
- ストレスレベル
- 心拍数連携データ

**スコア化方法**：
```
EmotionalControl_Score = (
  (100 - 感情的トレード率) × 40 +
  (100 - FOMO傾向スコア) × 40 +
  (ストレス管理スコア) × 20
)
```

**満点**：100点

**どのランク/クラス/称号で重視するのか**：
- チャレンジャー：重視（15%）
- BNF到達者：重視（15%）
- メンタルマスター称号：最重視

### 5. コンシステンシー（継続性・安定性）

**何を測るのか**：
- 月次勝率の安定性
- 月次損失月の少なさ
- 連勝/連敗の傾向
- 相場環境での適応性

**スコア化方法**：
```
Consistency_Score = (
  (月次勝率の平均) × 40 +
  (月次勝率の安定性) × 30 +
  (相場環境適応性) × 30
)
```

**満点**：100点

**どのランク/クラス/称号で重視するのか**：

- ブラックロック・クラス：重視（20%）
- コンシステント称号：最重視

### 6. リサーチ / レビュー（検証・振り返り）

## 公開/非公開の分け方

- 通常ランク（ブロンズ～チャレンジャー）

- シークレット称号の一部（ザ・ワン、ドローダウンゼロ、キャピタルダブル）
- 裏称号（全て）

- 裏称号（全て）
- 一部のシークレット称号

## まとめ

BRAKE Proの称号システムは、以下の特徴を持つ包括的な設計です：

1. **7層構造**で、異なる目的・価値観の称号を整理
2. **金融エリート感**を重視し、ゲーム的な安っぽさを排除
3. **規律・感情統制・継続性**を最重視し、「本当に格のあるトレーダー」を可視化
4. **6つの評価軸**で、多角的にトレーダーを評価
5. **公開/非公開の明確な分け方**で、プライバシーとステータスのバランスを取る
6. **ジョーク称号は改善導線付き**で、自虐ネタ＋改善を促す仕組み
7. **実装可能な数値条件**で、プロダクト化が容易

このシステムにより、BRAKE Proは単なるトレード管理ツールから、**トレーダーの成長と格を可視化するプラットフォーム**へと進化します。
